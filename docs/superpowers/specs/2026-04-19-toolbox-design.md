# 線上工具箱 設計規格

## 概覽

供刑事警察單位同仁使用的線上工具箱，提供偵查作業常用的數位工具。網頁版部署於 GitHub Pages，無需安裝；圖片壓縮工具另提供 Python 打包的本機版。

---

## 技術棧

| 類型 | 技術 |
|---|---|
| 前端框架 | 純 HTML + CSS + Alpine.js（4KB，無需建置工具） |
| PDF 處理 | pdf-lib（操作）+ PDF.js（縮圖預覽） |
| 圖片處理（網頁版）| heic2any + browser-image-compression + JSZip |
| 圖片處理（本機版）| Python + Pillow + pillow-heif + tkinter |
| 打包工具 | PyInstaller（Windows .exe + Mac .app） |
| 部署 | GitHub Pages（靜態網站，零後端） |

## PDF 工具（tools/pdf.html）

### 共用 UI
- 頁面頂部四個功能標籤：合併 ｜ 分割 ｜ 旋轉 ｜ 加密/解密
- Alpine.js 控制標籤切換，不跳頁

### 合併
1. 上傳多個 PDF（多選或拖曳）
2. 顯示清單，每個檔案呈現第一頁縮圖 + 檔名 + 頁數
3. 可拖曳或用上移/下移按鈕調整順序
4. 按「合併並下載」輸出新 PDF

### 分割
1. 上傳單一 PDF
2. 顯示所有頁面縮圖（每頁一格）
3. 點擊縮圖切換勾選狀態（保留/不保留）
4. 按「儲存選取頁面」下載只含勾選頁的新 PDF
5. 未勾選的頁面不納入輸出檔

### 旋轉
1. 上傳 PDF
2. 顯示所有頁面縮圖
3. 每頁旁有「順時針旋轉 90°」按鈕（可多次點擊，點四次回到原方向）
4. 確認後下載旋轉後的 PDF

### 加密/解密
1. 上傳 PDF
2. 選擇操作：加密 或 解密
3. 輸入密碼（解密時需輸入原密碼）
4. 下載處理後的 PDF

## 圖片壓縮工具

### 網頁版（tools/image.html）

**流程：**
1. 選取資料夾或拖曳多張照片
2. 輸入目標大小（MB，使用者自填）
3. 按「開始轉換」
4. 顯示處理清單：每張照片原始大小 → 壓縮後大小
5. 全部完成後提供 ZIP 下載

**規格：**
- 輸入格式：JPG、PNG、HEIC
- 輸出格式：JPG（統一）
- 所有處理在瀏覽器本地執行，不上傳伺服器
- 大批量（50 張以上）處理速度較慢，介面顯示進度提示

### 本機版（Python）

**流程：**
1. 開啟程式（tkinter 視窗）
2. 按鈕選擇來源資料夾
3. 按鈕選擇輸出資料夾（預設為來源資料夾旁的 `converted_photos/`）
4. 輸入目標 MB 大小
5. 按「開始轉換」，顯示進度條
6. 完成後顯示成功訊息與輸出位置

**規格：**
- 輸入格式：JPG、PNG、HEIC
- 輸出格式：JPG
- Windows 打包：`圖片壓縮工具.exe`
- Mac 打包：`圖片壓縮工具.app`
- 兩個平台需分別在對應系統上執行 PyInstaller 打包

## 未來擴充

新增工具只需：
1. 在 `tools/` 建立新的 HTML 頁面
2. 在 `index.html` 首頁新增一張卡片

不影響現有工具，Alpine.js 結構支援無限擴充。
